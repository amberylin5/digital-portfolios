AMS - Appointment Management System

Technologies: React.js, Node.js, MySQL

Category: Full-stack Web Application

Date: March 2025

Project Overview

AMS is a comprehensive e-commerce system designed for small businesses. It features a responsive admin dashboard, customer booking interface, and automated notifications. The system helps businesses manage appointments efficiently while providing a seamless booking experience for customers.

Key Features

  • Role-based authentication (Admin, Staff, Customer)
  • Interactive calendar with drag-and-drop scheduling
  • Automated email/SMS reminders
  • Real-time availability checking
  • Reporting and analytics dashboard

Technical Implementation

The frontend was built with React.js using Material-UI components for a consistent look and feel. The backend API was developed with Node.js and Express, with MySQL for data persistence. Key challenges includes implementing conflict-free scheduling and optimizing database queries for the calendar view.
